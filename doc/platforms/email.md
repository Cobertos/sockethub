# PLATFORM: email

## overview

The email platform currently supports both the SMTP (for sending) and IMAP (for
fetching) protocols. The current end-user verbs it supports are `send` and
`fetch`.


## `set` credentials



## examples

### sending an email


```javascript
  {
    rid: 123,
    platform: "email",
    verb: "send",
    actor: {
      name: "Me Foobar",
      address: "me@foobar.com"
    },
    target: [
      {
        field: "to",
        name: "Test Recipient",
        address: "test@example.com"
      },
      {
        field: "cc",
        name: "Other Recipient",
        address: "other@example.com"
      }
    ],
    object: {
      subject: "A test message",
      text: "This is a test message for Sockethub!",
      html: '<div><div>This is a <b>test message</b> for <font size="4">Sockethub!</font></div></div>'
    }
  }
```
